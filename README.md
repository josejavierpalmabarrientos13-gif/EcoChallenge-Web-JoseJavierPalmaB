# EcoChallenge-Web-JoseJavierPalmaB
## 🌍 Hola a todos, esta es la idea de mi proyecto:

> **Nombre del Proyecto:** EcoChallenge: El Juego del Cambio Climático
> 
> **Qué problema resuelve?**
> La falta de hábitos sostenibles cotidianos y la apatía de las personas frente a las pequeñas acciones individuales que pueden frenar el cambio climático.
> 
> **Cómo lo resuelve?**
> A través de un sitio web interactivo desarrollado con Python y Flask que gamifica la ecología. Cada día, la aplicación presenta al usuario un "Eco-Desafío" aleatorio. Si el usuario lo cumple, acumula puntos y sube de rango, transformando la responsabilidad ambiental en un juego diario y competitivo.

## Este es el tipo de mi proyecto:
> Una aplicación web interactiva (Game Web App) enfocada en la gamificación del cuidado ambiental.

## Las bibliotecas que voy a utilizar:
- **Flask (v3.0.0):** Micro-framework de Python para gestionar las rutas de la aplicación, peticiones HTTP (POST/GET) y la lógica de los puntos del juego.
- **Random (Nativa de Python):** Para la selección aleatoria de los ecodesafíos diarios.

## Referencias útiles:
- [Repositorio Base de Flask](https://github.com/pallets/flask) - Código fuente oficial y ejemplos base de aplicaciones web estructuradas con Flask.
- [Python Web Server Template](https://github.com/realpython/flask-boilerplate) - Estructura de referencia para proyectos limpios que conectan lógica de Python con plantillas HTML.
